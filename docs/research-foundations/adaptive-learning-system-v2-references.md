# Adaptive Learning System V2 — Research Foundations and References

This research set supports the documentation-level foundation for Adaptive Learning System V2 and is organized for future credibility, transparency, and implementation alignment.

## 1) Mastery Learning

- **Guskey, T. R.** (2010). *Lessons of Mastery Learning*. Phi Delta Kappan.
  - Core relevance: clarifies mastery learning as a system requiring clear outcomes, formative feedback, and support before advancement.
- **Bloom, B. S.** (1968). *Learning for Mastery*. Evaluation Comment.
  - Core relevance: foundational conceptual frame for mastery progression and corrective instruction.

## 2) Learning Trajectories

- **Sarama, J., & Clements, D. H.** (2009). *Learning and Teaching Early Math: The Learning Trajectories Approach*. Routledge.
  - Core relevance: establishes trajectory-based sequencing from current understanding to target competencies.
- **Opfer, V. D., & Pedder, D. G.** (2011). *Conceptualizing Teacher Professional Learning*. Review of Educational Research.
  - Core relevance: contributes systems-level framing for how learning progressions are interpreted and implemented across instructional settings.

## 3) Literacy Science

- **Pearson, P. D.** (selected body of work on reading comprehension and literacy development).
  - Core relevance: emphasizes comprehension development and literacy structures necessary for reading proficiency.
- **Evenson, E.** (literacy development and instructional practice references as curated for program alignment).
  - Core relevance: practical literacy progression considerations for early-grade readers.

## 4) Standards-Based Progression

- **Daro, P., Mosher, F. A., & Corcoran, T. B.** (2011). *Learning Trajectories in Mathematics: A Foundation for Standards, Curriculum, Assessment, and Instruction*. CPRE.
  - Core relevance: links standards expectations to coherent instructional pathways and assessment alignment.
- **Ainsworth, L.** (2010). *Rigorous Curriculum Design*. Lead + Learn Press.
  - Core relevance: supports standards deconstruction and coherent progression design.

## 5) Curriculum Design

- **Fonger, N. L.** (curriculum design and learning progression contributions in K–12 instructional systems).
  - Core relevance: practical scaffolding principles and sequence coherence for curriculum architecture.
- **Wiggins, G., & McTighe, J.** (2005). *Understanding by Design* (Expanded 2nd ed.). ASCD.
  - Core relevance: backward design structure for aligning goals, evidence, and learning experiences.
