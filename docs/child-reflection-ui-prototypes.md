# Child Reflection UI Prototype Concepts (Static / Design-only)

## Prototype A — Gate Reflection Entry Card

```
+---------------------------------------------------+
| Gate 4: Valley of Weather                         |
|---------------------------------------------------|
| "Today the valley sky is changing..."            |
|                                                   |
| [ 🌧 Rain ] [ 🌤 Sun ] [ 🌫 Fog ] [ 🌬 Wind ] [ ⛈ Storm ] |
|                                                   |
| Prompt: "What helps your weather soften?"        |
|                                                   |
| [ I breathe ] [ I ask for help ] [ I take space ] |
|                                                   |
| [ Continue Story ]         [ Pause for now ]      |
+---------------------------------------------------+
```

Notes:
- No score, no correctness language.
- Always includes `Pause for now`.
- Visual symbols before text for accessibility.

## Prototype B — Story Branch Reflection

```
Scene: The Bridge Builders

A bridge plank breaks while others are crossing.

Choose a path:
1) Watch quietly first
2) Ask who needs help
3) Start repairing with a friend
4) Gather tools for everyone

Reflection card after choice:
"What made that choice feel right today?"
```

Notes:
- Choice framed as perspective, not correctness.
- Follow-up prompt captures meaning-making language.

## Prototype C — Parent Bridge Card (post-session)

```
This week’s reflection themes:
- Emerging preference: asks for help during challenge
- Current pattern: pauses before reacting in conflict scenes
- Growth opportunity: naming feelings earlier

Suggested family conversation:
"What helps each of us feel safe when emotions get big?"
```

Notes:
- Non-diagnostic and non-labeling vocabulary.
- Family practice continuation anchor.

## Prototype D — Low-Stimulation Safety Shell

- Soft neutral palette.
- Minimal motion.
- One interaction cluster per screen.
- Optional audio narration toggle.
- Clear “Pause” and “Exit safely” affordances.
