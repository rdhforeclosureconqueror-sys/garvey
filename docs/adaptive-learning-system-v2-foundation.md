# Adaptive Learning System V2
## Phase 1A — Standards + Parent Roadmap Foundation

Strong mastery systems begin with:

- learning progressions
- grade-level expectations
- transparent mastery definitions
- standards-aligned pathways
- parent-readable roadmaps

This system foundation contains four permanent layers.

## Layer 1 — Grade Expectations

**Purpose:**

What is a child nationally expected to know by the end of a grade?

This reflects school expectations and standards.

Not internal engine logic.

## Layer 2 — Adaptive Skills Map

**Purpose:**

What skills teach those expectations?

**Structure:**

Grade  
→ Domain  
→ Skill  
→ Subskill  
→ Lessons  
→ Questions  
→ Mastery

## Layer 3 — Parent Roadmap

**Purpose:**

Where is my child now?

Parent visibility should eventually include:

- expected learning
- mastered skills
- developing skills
- next learning step

## Layer 4 — Mastery Definition

Mastery means:

accurate  
+  
consistent  
+  
independent  
+  
transferable

### Mastery bands

| Band | Score Range |
|---|---:|
| Emerging | 0–39 |
| Developing | 40–69 |
| Near Mastery | 70–84 |
| Mastered | 85–100 |

## Grade 1 Master Roadmap (V1)

### Parent Summary

By the end of Grade 1 most children should:

- read simple texts independently
- understand basic stories and information
- recognize high-frequency words
- decode simple words
- write complete simple sentences
- understand basic grammar

- add and subtract within 20

- compare numbers

**Grade Success Profile.**

# Grade 2 Master Roadmap (V1)

Grade 2 is a consolidation and bridge year.

Grade 1 builds basics.
Grade 2 builds automaticity and independence.
Grade 3 shifts toward reading to learn and more complex math.

## Grade 2 Year Profile

By the end of Grade 2, most children should:

Reading / English:
- read fluently and accurately
- understand stories and informational text
- answer questions using evidence
- build vocabulary
- write organized paragraphs
- apply grammar and punctuation correctly

Math:
- add and subtract within 100
- understand place value deeply
- begin multiplication foundations
- solve word problems
- work with measurement and money
- understand shapes and data

This becomes the Grade 2 Success Profile.

Domain 1: Number Sense + Place Value
