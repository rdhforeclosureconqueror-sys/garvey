# Adaptive Learning System V2
## Phase 1A — Standards + Parent Roadmap Foundation

Strong mastery systems begin with:

- learning progressions
- grade-level expectations
- transparent mastery definitions
- standards-aligned pathways
- parent-readable roadmaps

This system foundation contains four permanent layers.

## Layer 1 — Grade Expectations

**Purpose:**

What is a child nationally expected to know by the end of a grade?

This reflects school expectations and standards.

Not internal engine logic.

## Layer 2 — Adaptive Skills Map

**Purpose:**

What skills teach those expectations?

**Structure:**

Grade  
→ Domain  
→ Skill  
→ Subskill  
→ Lessons  
→ Questions  
→ Mastery

## Layer 3 — Parent Roadmap

**Purpose:**

Where is my child now?

Parent visibility should eventually include:

- expected learning
- mastered skills
- developing skills
- next learning step

## Layer 4 — Mastery Definition

Mastery means:

accurate  
+  
consistent  
+  
independent  
+  
transferable

### Mastery bands

| Band | Score Range |
|---|---:|
| Emerging | 0–39 |
| Developing | 40–69 |
| Near Mastery | 70–84 |
| Mastered | 85–100 |

## Grade 1 Master Roadmap (V1)

This is the first Grade Success Profile.

### Parent Summary

By the end of Grade 1 most children should:

#### Reading / English

- read simple texts independently
- understand basic stories and information
- recognize high-frequency words
- decode simple words
- write complete simple sentences
- understand basic grammar

#### Math

- understand numbers and place value
- add and subtract within 20
- solve simple word problems
- compare numbers
- measure simple objects
- recognize shapes and patterns

This becomes:

**Grade Success Profile.**

## Grade 1 Math Roadmap

### Domain 1: Number Sense + Counting

**Skills:**

- count forward
- count backward
- count to 120
- number order
- compare numbers
- greater/less/equal

**Parent Goal:**
Understand how numbers work.

### Domain 2: Place Value

**Skills:**

- tens and ones
- represent numbers
- compose numbers
- decompose numbers

**Parent Goal:**
Understand numbers are built from tens and ones.

### Domain 3: Addition + Subtraction

**Skills:**

- facts within 10
- facts within 20
- addition strategies
- subtraction strategies
- equations
- missing numbers

**Parent Goal:**
Solve basic math accurately.

### Domain 4: Word Problems

**Skills:**

- add story problems
- subtract story problems
- identify math language
- choose operations

**Parent Goal:**
Use math in real situations.

### Domain 5: Measurement + Time

**Skills:**

- compare length
- measure objects
- tell time basics
- understand calendars

**Parent Goal:**
Measure and organize daily life.

### Domain 6: Geometry

**Skills:**

- 2D shapes
- 3D shapes
- attributes
- patterns

**Parent Goal:**
Recognize and describe shapes.

## Grade 1 Reading / English Roadmap

### Domain 1: Foundational Reading

**Skills:**

- letter-sound knowledge
- phonemic awareness
- blending
- decoding
- sight words

**Parent Goal:**
Read words independently.

### Domain 2: Reading Fluency

**Skills:**

- pacing
- accuracy
- expression
- repeated reading

**Parent Goal:**
Read smoothly.

### Domain 3: Comprehension

**Skills:**

- story sequence
- characters
- setting
- key details
- retelling

**Parent Goal:**
Understand what was read.

### Domain 4: Vocabulary

**Skills:**

- context clues
- word meaning
- categories
- descriptive language

**Parent Goal:**
Build language power.

### Domain 5: Writing + Grammar

**Skills:**

- sentence formation
- capitalization
- punctuation
- nouns
- verbs
- simple writing

**Parent Goal:**
Express ideas clearly.
