# G2E_RC_003 — Main Idea and Key Details Curriculum Review

## Scope and method

Every one of the 50 questions in the four focused banks and the Mixed bank was read in full. The review checked passage meaning, Grade 2 language, the relationship between each main idea and its details, answer uniqueness, distractor plausibility, explanations, interaction support, and duplication. Mixed review now uses ten new passages rather than recycling focused practice.

## Representative educational improvements

### `G2E_RC_003_LVL1_Q01`

- **Issue found:** The original topic item was duplicated later and paired the topic with a main idea, one detail, and the label “one tiny example.”
- **Educational reason:** Those choices were not parallel topic choices, so a learner could answer from grammar rather than comprehension.
- **Improvement:** A new passage about ant colonies now offers four believable noun topics. The explanation distinguishes the passage-wide topic from other subjects.

### `G2E_RC_003_LVL2_Q03`

- **Issue found:** The old bank reused the same five passages and used the vague distractor “Everything in the world is connected.”
- **Educational reason:** A distractor should reveal whether a learner can choose an idea supported by all details, not whether the learner can spot exaggerated wording.
- **Improvement:** A seasonal maple-tree passage now has four complete-sentence main ideas. The correct answer covers spring, summer, and fall details, while each distractor stays on a related, age-appropriate subject.

### `G2E_RC_003_LVL3_Q04`

- **Issue found:** Incorrect options were labels such as “A fun opinion about the topic” and “A detail from another text.”
- **Educational reason:** Generic labels do not ask children to compare evidence from authentic sentences.
- **Improvement:** The new item states the main idea that Eli dressed for snowy weather. Learners compare four natural details and identify the clothing evidence that directly supports it.

### `G2E_RC_003_LVL4_Q06`

- **Issue found:** Mechanical arrow notation matched a detail to “an unrelated main idea” or “unsupported detail.”
- **Educational reason:** The notation and labels tested elimination of placeholders instead of connecting ideas across a passage.
- **Improvement:** A classroom welcome passage asks which detail shows kindness. All choices are facts connected to the same setting, but only the invitation to play supports the stated idea.

### `G2E_RC_003_MIXED_Q08`

- **Issue found:** Mixed questions repeated focused passages and answer sets exactly.
- **Educational reason:** Repetition can measure memory of an earlier answer instead of transfer of the main-idea strategy.
- **Improvement:** A new seed experiment asks learners to infer why light conditions differ while water stays the same. This applies key-detail reasoning in a fresh science context.

## Technical and interaction review

The package uses the existing `multiple_choice`, `short_response`, and `text_evidence` interactions. Focused validation exercises schema acceptance, renderer output, correct-answer evaluation, required passages, normalized choice uniqueness, placeholders, explanations, signature uniqueness, Mixed transfer, and authored answer-position balance. No shared runtime change was required.
