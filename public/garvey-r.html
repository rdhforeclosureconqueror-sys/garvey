R — Route

This phase turns identity into an execution path. The Kanban board is your official route map.

Route Rules (simple + professional)

  • If stuck: move card to Blocked and write what’s blocking it.
  • If you finish: move to Done immediately.
  • If it’s unclear: create a card titled “Decision Needed” and assign a due date (later).
No guessing. The board is the truth.
