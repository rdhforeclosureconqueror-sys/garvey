# GateQuest Developmental Game Platform MVP

This package turns the uploaded prototypes into one Gate-aligned system.

## Files

- `gatequest-standalone.html` — direct-open playable prototype
- `docs/10_GATE_GAME_MAP.md` — full Gate mapping
- `docs/CONTRACTOR_HANDOFF.md` — technical handoff for implementation

## How to run

Open `gatequest-standalone.html` in a browser.

## What it does

- Provides a child profile
- Lists all 10 Gates
- Launches Gate-mapped game modules
- Records developmental events
- Produces progress observations
- Exports a JSON report

## Current game modules

- Forest of Whispers
- NeuroSpark Gate Missions
- Brain Game Suite
- Resilience Breaker
- Social Quest Cards
