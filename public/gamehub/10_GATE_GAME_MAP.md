# 10-Gate Game Map

| Gate | Games | Mechanics |
|---|---|---|
| 1 Attention | Forest of Whispers, Distraction Defender, Visual Memory | Focus return, selective filtering, sustained observation |
| 2 Impulse Control | Freeze Runner, Plasma Hold, Rhythm Race, Resilience Breaker | Stop signal, wait gates, delayed response, steady movement |
| 3 Working Memory | Visual Memory, Forest Pattern Recall, Switch Matrix | Remember sequences, symbols, rules, and locations |
| 4 Emotional Regulation | Calm Reactor, Forest Recovery, Resilience Breaker | Breathing, calm relaunch, gentle correction |
| 5 Resilience | Resilience Breaker, Forest of Whispers | Return after mistakes, persistence, frustration tolerance |
| 6 Cognitive Flexibility | Switch Matrix, Picture Puzzle, Social Quest Cards | Rule switching, perspective shift, strategy change |
| 7 Planning & Sequencing | Picture Puzzle, Resilience Breaker, Visual Memory | Multi-step order, fewer random moves, sequence building |
| 8 Body Rhythm & Timing | Rhythm Race, Plasma Hold, Forest of Whispers | Timing consistency, rhythm synchronization, breath pacing |
| 9 Self-Awareness | Calm Reactor, Resilience Breaker, Social Quest Cards | Name state, notice strategy, reflect after challenge |
| 10 Social Awareness | Social Quest Cards | Perspective-taking, cooperation, repair, empathy choices |

## Core rule

A single game does not need to cover every Gate. The platform covers all 10 Gates by logging mechanic-level events across multiple games.
